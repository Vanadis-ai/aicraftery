# AI Craftery

Public release distribution for [AI Craftery](https://aicraftery.com) — workspace tool for layered AI-native development.

## Install

**macOS:**
```
curl -fsSL https://releases.aicraftery.com/install-mac.sh | bash
```

**Linux** (Ubuntu / Debian / Fedora / Arch / openSUSE):
```
curl -fsSL https://releases.aicraftery.com/install-linux.sh | bash
```

Or grab the platform installer from [Releases](https://github.com/Vanadis-ai/aicraftery/releases): `.pkg` for macOS, `.deb` / `.AppImage` / `.tar.gz` for Linux.

## What this repo is

This is the public mirror for AI Craftery release artifacts and signed update manifests. Source code lives in a private repository.
